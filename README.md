# 오브젝트 스터디

* 교재 : 오브젝트: 코드로 이해하는 객체지향 설계

* 스터디 규칙
    * 챕터마다 있는 코드를 직접 입력해본다.
    * 따라치는 것이 아닌 설명을 보고 먼저 코딩한 후 교재의 코드와 비교해본다.
    * 소챕터마다 commit 하며 리팩토링한다.
